# BMI
### A healthy android app.
This Project is about healthy body and it will calculate your BMI and some other useful information and return them in a perfect UI. Project is based on MVVM Structure and
 written in Kotlin.
### View
#### The app just include two page (in view): One for Input and One for Result.
In Input page we get user's weight and height and he/she have to choose his/her Gender and also if he/she is athlete, can check that. After this simple UI and with click 'Next' 
the user can see many data about his/herself. exp. BMI number - BMI status - Normal weight - Suggested calorie and etc.
### Model and ViewModel
#### Both Model and ViewModel are as easy as you think and they are so clear.
